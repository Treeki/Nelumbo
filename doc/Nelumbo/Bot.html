Nelumbo::Bot

The Nelumbo::Bot class implements a general-purpose bot for Furcadia.

Settings Manager

Settings can be assigned to a class and inherited by subclasses of that class, using the Bot::set method. They can be queried using Bot::setting and Bot::setting?.

Individual instances of a class can also have their own settings, which can be modified at any time and override (but do not replace) the class settings.

Connection

The Bot class handles logging in for you.

The :username and :password settings are not optional:

set username: 'Treeki', password: 'hunter2'

The :color_code and :description settings are optional, but recommended. Usage is fairly obvious.
