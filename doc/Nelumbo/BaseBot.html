Nelumbo::BaseBot

The Nelumbo::BaseBot class implements a simple bot that can communicate with Furcadia and handle events using Nelumbo::EventHandler.

Socket and timer handling is done using a Core, which can be passed to BaseBot#new. If none is specified, a new instance of Nelumbo::SimpleCore will be used.

Usage of this class directly is not recommended. Nelumbo::Bot implements plugins and various other niceties that are useful for most bots.
